# mhjavabase

这个开源旨在通过代码的形式回顾java的基础知识

里面所有的代码都是猿人在最初接触java的时候跟着老师一行一行手敲的代码，从这个开源开始重新回顾基础，在原来代码的基础之上重新理解和添加完整的注释。

内容适合初学者了积累代码量，理解基础知识
